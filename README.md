# Disney Webpage - Maquetación Profesional

Parcial II - Fundamentos de Programación Web.
Maquetación pixel-perfect de la página oficial de Disney usando únicamente **HTML5** y **CSS3** (sin JavaScript).

## 👥 Integrantes

- GojoSatorouXD
- Gabimaru23-A

## 🛠️ Tecnologías y metodologías aplicadas

- **HTML5 semántico** (`<nav>`, `<section>`, `<main>`, `<article>`, `<footer>`)
- **CSS3** con variables personalizadas (`:root`)
- **Modelo de caja** (`box-sizing: border-box`)
- **CSS Grid** para layouts bidimensionales (shop, movies, parks, more)
- **Flexbox** para componentes unidimensionales (navbar, banner, cards)
- **Metodología BEM** (`block__element--modifier`) para el nombrado de clases
- **Diseño responsive** con `@media queries` (desktop, tablet, mobile)
- **Viewport meta tag** para diseño responsive
- Idioma: **Inglés** (`lang="en"`)

maquetacion-profesional-web-page/

## 🧩 Secciones maquetadas

1. **Navbar** - Logo, menú de navegación, login y búsqueda
2. **Banner** - Hero de Snow White con call-to-actions
3. **Shop** - Grid de 4 cards promocionales
4. **Disney+** - Grid de 3 cards de contenido destacado
5. **Movies** - Grid de 5 próximos estrenos
6. **Parks & Travel** - Grid de 2 ofertas vacacionales
7. **More** - Sección de ofertas con tarjeta Visa
8. **Footer** - Navegación secundaria y copyright

## 📱 Breakpoints responsive

- **Desktop:** > 992px
- **Tablet:** ≤ 992px
- **Mobile:** ≤ 640px

## 🚀 Cómo ejecutar

Simplemente abrir `index.html` en cualquier navegador moderno.
